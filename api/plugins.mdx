## List Plugins

Get all available plugins. Non-admin users only see enabled plugins.

<CodeGroup>
```bash cURL
curl -X GET https://api.stellarstack.io/plugins \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

### Response

Returns an array of plugin objects.

<ResponseField name="pluginId" type="string">
  Unique plugin identifier
</ResponseField>

<ResponseField name="name" type="string">
  Plugin display name
</ResponseField>

<ResponseField name="version" type="string">
  Plugin version
</ResponseField>

<ResponseField name="description" type="string" required={false}>
  Plugin description
</ResponseField>

<ResponseField name="author" type="string" required={false}>
  Plugin author
</ResponseField>

<ResponseField name="status" type="string">
  Plugin status: "installed", "enabled", "disabled", "error"
</ResponseField>

<ResponseField name="category" type="string">
  Plugin category
</ResponseField>

<ResponseField name="isBuiltIn" type="boolean">
  Whether this is an official built-in plugin
</ResponseField>

## Get Plugin Details

Retrieve detailed information about a specific plugin.

<CodeGroup>
```bash cURL
curl -X GET https://api.stellarstack.io/plugins/{pluginId} \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

### Path Parameters

<ParamField path="pluginId" type="string" required>
  Plugin ID (e.g., "curseforge-installer")
</ParamField>

### Response

<ResponseField name="pluginId" type="string">
  Plugin ID
</ResponseField>

<ResponseField name="name" type="string">
  Plugin name
</ResponseField>

<ResponseField name="version" type="string">
  Current version
</ResponseField>

<ResponseField name="description" type="string" required={false}>
  Plugin description
</ResponseField>

<ResponseField name="status" type="string">
  Plugin status
</ResponseField>

<ResponseField name="config" type="object">
  Current plugin configuration
</ResponseField>

<ResponseField name="manifest" type="object">
  Full plugin manifest with actions, permissions, etc.
</ResponseField>

**404 Not Found** - Plugin not found

## Execute Plugin Action

Execute a plugin action on a server.

<CodeGroup>
```bash cURL
curl -X POST https://api.stellarstack.io/plugins/{pluginId}/actions/{actionId} \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "serverId": "server-id",
    "inputs": {
      "version": "1.20.4"
    },
    "confirmed": true
  }'
```
</CodeGroup>

### Path Parameters

<ParamField path="pluginId" type="string" required>
  Plugin ID
</ParamField>

<ParamField path="actionId" type="string" required>
  Action ID from plugin manifest
</ParamField>

### Request Body

<ParamField body="serverId" type="string" required>
  Server ID to execute action on
</ParamField>

<ParamField body="inputs" type="object">
  Action input parameters (schema defined by plugin action)
</ParamField>

<ParamField body="confirmed" type="boolean">
  Confirmation for dangerous actions (required if action is marked dangerous)
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Whether action executed successfully
</ResponseField>

<ResponseField name="executedOperations" type="number">
  Number of operations completed
</ResponseField>

<ResponseField name="error" type="string" required={false}>
  Error message if action failed
</ResponseField>

### Error Responses

**400 Bad Request**

<ResponseField name="error" type="string">
  Error message
</ResponseField>

<ResponseField name="requiresConfirmation" type="boolean">
  True if action requires confirmation
</ResponseField>

<ResponseField name="action" type="object">
  Action details
</ResponseField>

**403 Forbidden** - Access denied to server

**404 Not Found** - Plugin or action not found

## Get Plugin Storage

Get a value from plugin storage. **Admin only.**

<CodeGroup>
```bash cURL

  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

### Path Parameters

<ParamField path="pluginId" type="string" required>
  Plugin ID
</ParamField>

<ParamField path="key" type="string" required>
  Storage key
</ParamField>

### Query Parameters

<ParamField query="serverId" type="string">
  Optional server ID for server-scoped storage
</ParamField>

### Response

<ResponseField name="key" type="string">
  Storage key
</ResponseField>

<ResponseField name="value" type="any">
  Stored value
</ResponseField>

## Query Audit Log

Query the plugin audit log with filters. **Admin only.**

<CodeGroup>
```bash cURL
curl -X GET "https://api.stellarstack.io/plugins/audit?pluginId=plugin-id&limit=100" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

### Query Parameters

<ParamField query="pluginId" type="string">
  Filter by plugin ID
</ParamField>

<ParamField query="userId" type="string">
  Filter by user ID
</ParamField>

<ParamField query="serverId" type="string">
  Filter by server ID
</ParamField>

<ParamField query="result" type="string">
  Filter by result: "success", "error", "denied"
</ParamField>

<ParamField query="limit" type="number" default="100">
  Maximum number of entries to return
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of entries to skip
</ParamField>

### Response

Returns an array of audit log entries.

## Install Plugin from Git

Install a plugin from a Git repository. **Admin only.**

<CodeGroup>
```bash cURL
curl -X POST https://api.stellarstack.io/plugins/install \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "repoUrl": "https://github.com/username/plugin-name",
    "branch": "main"
  }'
```
</CodeGroup>

### Request Body

<ParamField body="repoUrl" type="string" required>
  Git repository URL
</ParamField>

<ParamField body="branch" type="string" default="main">
  Branch to install from
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Always true
</ResponseField>

<ResponseField name="plugin" type="object">
  Installed plugin object
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

**400 Bad Request** - Installation failed

## Get Security Report

Get the security analysis report for a plugin. **Admin only.**

<CodeGroup>
```bash cURL
curl -X GET https://api.stellarstack.io/plugins/{pluginId}/security-report \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

### Path Parameters

<ParamField path="pluginId" type="string" required>
  Plugin ID
</ParamField>

### Response

<ResponseField name="pluginId" type="string">
  Plugin ID
</ResponseField>

<ResponseField name="name" type="string">
  Plugin name
</ResponseField>

<ResponseField name="trustLevel" type="string">
  Trust level: "official" or "community"
</ResponseField>

<ResponseField name="securityScore" type="number">

</ResponseField>

<ResponseField name="securityReport" type="object">

      Security score
    </ResponseField>
    <ResponseField name="riskLevel" type="string">
      Risk level assessment
    </ResponseField>

      Security issues found
    </ResponseField>
    <ResponseField name="warnings" type="array">
      Security warnings
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="analyzedAt" type="string">
  ISO 8601 timestamp of last analysis
</ResponseField>

**404 Not Found** - Plugin not found
